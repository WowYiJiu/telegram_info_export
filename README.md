# Telegram_Info_Export

- 导出群组和频道（有邀请链接和无邀请链接分开显示）
- 导出机器人
- 导出对话中的个人用户（不包含无用户名的）

### 使用说明

> 使用时需关掉 2FA

- `api_id`和`api_hash`需要到[Telegram 开发者平台](https://my.telegram.org/auth)申请
- proxy_url 格式

```
socks5://wowyijiu:wowyijiu@127.0.0.1:23333
socks5://127.0.0.1:23333
http://127.0.0.1:7890
```
